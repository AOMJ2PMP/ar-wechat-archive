CodeBuddy Code (CLI Agent) v1.3 全新升级🚀

💻 快速开始：  npm install -g @tencent-ai/codebuddy-code

请看本次全新升级重点⬇️
✅ 效能再进化：支持 Sub-Agents 协同工作、新增 /agents 和 /memory 管理命令，内置 Jupyter 笔记本，极大地提升了开发效率和无缝衔接体验
