# CodeBuddy Code 2.0 重磅升级！

⬇️本次升级重点：

✨开放 SDK 被集成，为业务注入 AI Agent 新动力！
通过 Plan 模式，让自然语言对话编程更透明，ACP协议实现Agent能力标准化输出，更易协作与集成；通过 CodeBuddyAgent SDK 和 APIKey 方式，加速使企业和开发者即刻以最低成本构建生产级部署的 AI Agent 应用，腾讯1.2万人、15+核心业务深度使用与集成。

✨兼容开发者社区生态建设与集成
作为国内最早支持 Skills 技能能力的编程助手，让 AI 具备现实世界人的技能。同时，面对终端开发者对效率提升场景的迫切需求，我们全面拥抱开发者社区工具生态，支持 Plugin 插件市场，满足开发者贡献插件及自定义 Subagents 智能体、一键自定义指令触发生成和 AgentHooks 管家，满足用户低成本高效迁移 CLl Agent 生态内容，提升研发效率。

✨支持隔离沙箱环境安全运行，集成 TencentOS，构建安全可靠的 AI Infra 新基建
支持 CodeBuddy Code 在 Container/E2B 隔离沙箱环境中进行安全运行安全的代码执行和项目开发环境，同时支持沙箱化 Bash 工具来强制执行文件系统和网络隔离，让其在安全的沙箱中更自主地工作。

✨增强优化多项功能
我们对上百万代码进行工程记忆及理解增强、交互模式、工具权限及安全等能力进行体验升级，支持自定义模型及 GLM-4.7/GPT5.2Codex 等国内外高级模型。从需求规划到工程深度理解与强化记忆、 LSP 代码智能与诊断能力，到制作和上架 Plugin 插件市场及一键交付，同时上线国内企业版上线，支持腾讯统一身份认证，研效度量，安全审计，组织知识资产托管及分发，全面为终端开发者/开发者团队提供高质量代码结果和企业级 AI 应用解决方案，构建极致沉浸式开发体验。

🎯全流程落地 AI Native 团队多 Agent 并行开发新范式
CodeBuddy Code 团队通过自举 AI 提效实践，转化 AI 为主角，构建面向未来编程的 AI 开发团队，AI 实现软件研发全流程闭环，落地 AI Native 团队开发新范式，近 4 月里，CodeBuddy Code实现近 100% 代码由 AI 自生成。

🚀无论是快速搭建 AI Agent 应用，还是构建安全高效的开发流程，CodeBuddy Code 都将成为 AI Native 开发时代的必备工具！

👉"码"力全开，即刻免费体验，安装指引： https://copilot.tencent.com/docs/cli/overview
