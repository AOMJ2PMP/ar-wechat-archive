# 插件升级 3.3丨一个对话框无缝衔接创作与问答

官网地址：https://copilot.tencent.com/

插件 3.3 版本核心功能升级一览👇：

1️⃣ Craft / Ask 升级融合协作，一个对话框无缝衔接创作与问答
2️⃣ 支持更全面「查看变更」功能，集成差异高亮与历史对
3️⃣ 支持对代码改动的「回退改动」操作
4️⃣ 新增@下选择「文件夹」能力，实现对特定目录进行直接代码操作，大幅减少文件检索与切换耗时
5️⃣ 新增代码审查「/cr」、单元测试「/tests」快捷指令
6️⃣ 新增「shift + 拖拽」快捷键拖拽文件至工作
