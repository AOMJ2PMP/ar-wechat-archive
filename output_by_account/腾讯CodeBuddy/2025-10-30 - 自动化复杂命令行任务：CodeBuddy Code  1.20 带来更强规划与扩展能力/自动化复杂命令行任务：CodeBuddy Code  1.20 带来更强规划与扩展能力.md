# 自动化复杂命令行任务：CodeBuddy Code 1.20 带来更强规划与扩展能力

CLI Agent 更新发布 1.20，本次升级重点围绕智能体核心推理能力、模型基础与系统可扩展性，为专业开发者提供更可靠、更强大的终端命令行自动化伙伴

💻 安装命令：npm install -g @tencent-ai/codebuddy-code

✨核心更新亮点​
1️⃣ ​更强的任务规划与推理能力：Plan & Think 模式​。
显著提升了处理多步骤、有依赖关系的命令行任务的可靠性
​2️⃣ 模型内核升级：支持 GPT-5 - Codex​。
让代码理解、生成与推理全面进化
​3️⃣ 构建可控的智能工作流：支持规约编程、Hooks 与 Skills 技能。
通过 Speckit / OpenSpec 定义开发流程，用 Skills 封装可复用能力，再借 Hooks 嵌入自动化脚本，提供了不同维度的定制与扩展性
​4️⃣ 丰富的内置指令集：​​ 提供了 /statusline、/todo 等更多开箱即用的快捷命令
​
🤖 欢迎更新 CodeBuddy Code 至最新版本，探索更智能的命令行自动化！
