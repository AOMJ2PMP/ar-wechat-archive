# CodeBuddy 4.0 重磅发布丨现在，亲手定义你的专属开发 Buddy

CodeBuddy IDE 4.0 现已全面上线，此次更新聚焦 自定义智能体、记忆增强、项目级 MCP 配置与模型能力提示，只为给你带来更懂你、更顺手、更个性化的编程伙伴体验。

🎯 核心功能亮点

1️⃣ 自定义智能体 Sub-Agents 上线
可以按需创建专属 Agent，让每个智能体分工明确，轻松应对不同任务，还支持在团队与项目间灵活复用，效率直接拉满
🔗 详细指引：
https://cnb.cool/codebuddy/codebuddy/-/blob/master/docs/ide-zh/Subagents.md
2️⃣ Memory 功能
自动保存你的操作习惯、项目上下文与关键信息，每一次对话都接得住梗、续得上思路。
3️⃣ 支持项目级 MCP 配置
可在项目根目录中配置 .mcp.json 文件，实现更灵活的项目级别配置管理
4️⃣ 新增模型能力提示
鼠标悬停即可清晰查看模型支持的各类能力说明，帮助你决策更从容

🤖 立即体验 CodeBuddy IDE 4.0，定制真正懂你的 AI 开发伙伴，开启更个性化、更丝滑的编码协作新方式。
https://www.codebuddy.ai/
